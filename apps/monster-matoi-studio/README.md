# モンスター纏いスタジオ

ファンタジーのモンスターを「身に纏う」キャラクターのAI画像生成プロンプトを作るWebアプリです。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く（インターネット接続不要）

### STEP 1: ベースキャラ画像
- キャラクターの画像をドラッグ&ドロップ、またはクリックして選択（任意）
- JPEG / PNG / WebP、10MB以内
- 「キャラの特徴メモ」に銀髪・赤い瞳などの特徴を書くとプロンプトに反映される
- 画像はブラウザ内でプレビューのみ。どこにも送信されない

### STEP 2: モンスターを選ぶ
- カテゴリタブや検索ボックスで絞り込む（24体収録）
- カードをクリックすると詳細（伝承・纏い方・ポーズ）が表示される

### STEP 3: 画風を選ぶ
- **ちびキャラ・グロッシー3D風** — 映画のような奥行きのある背景
- **トレーディングカード風** — 装飾フレームとタイポグラフィ
- **水彩絵本風** — 細い線画とやわらかな水彩タッチ

### STEP 4: オプション
- **纏い度**：ワンポイント / しっかり纏う / 完全融合 から選ぶ
- **Midjourneyパラメータ**：チェックするとプロンプト末尾に `--niji 6 --ar XX:XX` が付く

### STEP 5: プロンプトを生成・コピー
- 「プロンプトを生成」ボタンを押す
- 英語プロンプト（画像生成AI用）と日本語版（確認用）が表示される
- 「コピー」ボタンでクリップボードにコピー

### 画像生成AIへの渡し方
Midjourney / nijijourney / NanoBanana 等で使う場合：
- ベースキャラ画像を「参照画像」として添付
- コピーした英語プロンプトを貼り付けて生成

## 収録モンスター（24体）

| カテゴリ | モンスター |
|---|---|
| 定番 | スライム、ミミック |
| 竜 | ドラゴン（西洋） |
| 幻獣 | フェニックス、グリフォン、ユニコーン |
| 水棲 | クラーケン、リヴァイアサン、セイレーン |
| 精霊 | ゴーレム（岩）、イフリート、ウンディーネ、九尾の狐、トレント |
| 魔獣 | ケルベロス、フェンリル、ベヒーモス、キマイラ、ヒュドラ、バジリスク、ミノタウロス、メデューサ、ハーピー |
| アンデッド・怪異 | ウィルオウィスプ |
